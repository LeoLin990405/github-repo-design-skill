# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [2.0.0] - 2026-03-16

### Added
- Phase-based module routing (Setup, Documentation, Collaboration, Quality, Advanced)
- Intent-based auto-routing for user requests
- 4 executable templates: new-repo-checklist, readme-template, ci-workflow-template, release-checklist
- Project-type recipes for module combinations
- CONTRIBUTING.md with contribution guidelines
- CHANGELOG.md to track project history
- GitHub issue templates (bug report, feature request)
- Pull request template
- Restructured README with badges, feature table, and project structure

### Changed
- Reorganized 15 standalone modules into 5 phase-grouped workflows
- Upgraded from knowledge-only guides to actionable templates
- Replaced linear difficulty levels with phase-based routing

## [1.0.0] - 2024-01

### Added
- Initial release with 15 knowledge modules
- Modules covering structure, readme, config, templates, settings, CI/CD, profile, security, monorepo, docs, versioning, quality, community, i18n, and FAQ
- SKILL.md entry point for Claude Code integration
